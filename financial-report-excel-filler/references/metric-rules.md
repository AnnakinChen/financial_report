# 财务指标口径参考

仅当 Excel 模版要求计算常见财务指标，且模版没有给出明确公式、用户也没有另行指定公式时，才参考本文件。若指标名称、期间、分母分子、平均数口径或归母口径存在歧义，应询问用户；无法明确时填 `-`。

## 通用约定

- 同一指标的所有组成项必须使用相同年度、币种、单位和报表口径。
- 涉及资产负债表平均余额的指标，只有在期初和期末余额都可核验时才计算。
- 如果模版已有百分比格式，应写入小数形式的比率，并保留百分比格式。
- 如果模版已有示例值显示为“百分数数值”而不是百分比格式，应跟随模版已有格式。
- 任一必要组成项缺失、无法核验或口径不一致，结果填 `-`。
- 净现金流指的是现金流量表里的现金及现金等价物净增加额

## 盈利能力指标

| 指标 | 公式 | 注意事项 |
| --- | --- | --- |
| 毛利率 | （营业收入 - 营业成本）/ 营业收入 | 营业收入和营业成本均需可核验。 |
| 营业利润率 | 营业利润 / 营业收入 | 不得用 EBIT 替代，除非模版明确如此要求。 |
| 净利率 | 净利润 / 营业收入 | 默认使用合并口径净利润；若模版要求归母口径，使用归母净利润。 |
| 总资产收益率 / ROA | 净利润 / 平均资产总额 | 需要期初和期末资产总额。 |
| 净资产收益率 / ROE | 净利润 / 平均所有者权益 | 若要求归母 ROE，应使用归母净利润和归属于母公司所有者权益。 |

## 偿债能力和杠杆指标

| 指标     | 公式                             | 注意事项                 |
|--------|--------------------------------|----------------------|
| 资产负债率  | 负债合计 / 资产总计                    | 使用总负债和总资产。           |
| 权益比率   | 所有者权益合计 / 资产总计                 | 使用所有者权益合计。           |
| 流动比率   | 流动资产合计 / 流动负债合计                | 不得用总资产或总负债替代。        |
| 速动比率   | （流动资产合计 - 存货）/ 流动负债合计          | 存货缺失时填 `-`。          |
| 利息保障倍数 | 息税前利润 / 利息费用                   | 只有 EBIT 和利息费用明确时才计算。 |
| 短期贷销比  | （短期借款+应付票据+一年内到期的非流动负债） / 营业收入 | 营收若为月度、季度数据，需年化      |

## 营运能力指标

| 指标 | 公式 | 注意事项                         |
| --- | --- |------------------------------|
| 总资产周转率 | 营业收入 / 平均资产总额 | 需要平均资产，除非模版明确使用期末资产。         |
| 存货周转率 | 营业成本 / 平均存货 | 需要期初和期末存货。营业成本若为月度、季度数据，需要年化 |
| 应收账款周转率 | 营业收入 / 平均应收账款 | 需要期初和期末应收账款。营业成本若为月度、季度数据，需要年化                |
| 应付账款周转率 | 营业成本 / 平均应付账款 | 需要期初和期末应付账款。营业成本若为月度、季度数据，需要年化                 |

## 成长能力指标

| 指标 | 公式 | 注意事项 |
| --- | --- | --- |
| 营业收入增长率 | （本年营业收入 - 上年营业收入）/ 上年营业收入 | 上年数据必须可核验。 |
| 净利润增长率 | （本年净利润 - 上年净利润）/ 上年净利润 | 上年数据必须可核验。 |
| 总资产增长率 | （本期期末资产总计 - 上期期末资产总计）/ 上期期末资产总计 | 上期期末数据必须可核验。 |
| 复合增长率 / CAGR | （期末值 / 期初值）^(1 / 年数) - 1 | 只有起止期间和年数明确时才计算。 |

## 现金流指标

| 指标   | 公式                    | 注意事项                               |
|------|-----------------------|------------------------------------|
| 收现比  | 销售商品、提供劳务收到的现金 / 营业收入 | 两项均需可核验。                   |
| 净现比  | 经营活动产生的现金流量净额 / 净利润   | 两项均需可核验。                           |
| 自由现金流 | 经营活动产生的现金流量净额 - 资本性支出 | 资本性支出必须明确披露；不得仅凭固定资产变动倒推，除非用户明确要求。 |


## 每股指标

只有当报告明确披露股本、加权平均普通股股数或模版提供明确公式时，才计算每股指标。

| 指标 | 公式 | 注意事项 |
| --- | --- | --- |
| 每股收益 / EPS | 归属于普通股股东的净利润 / 加权平均普通股股数 | 如报告已披露 EPS，优先使用披露值。 |
| 每股净资产 | 归属于母公司所有者权益 / 期末总股本 | 两个组成项都必须明确。 |
